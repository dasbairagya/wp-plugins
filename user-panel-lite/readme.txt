User Panel Lite is a responsive User Panel plugin for WordPress with which you could able to view all your users under a single roof.

== Description ==
The User Panel Lite plugin allows administrators to easily track all its users in one place. The User Panel Lite has been created via the WordPress administrative interface. 

= Docs & Support =

You can find [docs](https://github.com/filteraction/user-panel-lite) and more detailed information about User Panel Lite on [filteraction.com](https://www.filteraction.com/). If you can't locate any topics that pertain to your particular issue, post a new topic for it.

== Installation ==
1. Upload `user-panel-lite` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
