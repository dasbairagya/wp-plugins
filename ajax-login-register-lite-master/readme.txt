Ajax Login Register Lite is a responsive User Authentication plugin for WordPress with which you could able to Register/Login and build profile under a single roof.

== Description ==
Ajax Login Register Lite plugin allows user to easily register or login to their account and can edit their detils on the profile page. The Ajax Login Register Lite has been created via the WordPress administrative interface.

== Installation ==
1. Upload `Ajax Login Register Lite` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
