User Panel Lite is a responsive User Panel plugin for WordPress with which you could able to view all your users under a single roof.

== Description ==
The Subcat Listing Widget plugin allows administrators to easily list all subcategories of the current category. The Subcat Listing Widget has been created via the WordPress administrative interface.

== Installation ==
1. Upload `subcat-listing-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
